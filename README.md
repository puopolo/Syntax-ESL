# Syntax Constraint Grammar for ESL
For Syntax class by Jeremy Barnes (EHU/UPV HAPLAP 2023-2024)

The task was to create a constraint grammar targeting cross-linguistic transfer errors by Spanish speakers learning English. Directly relevant to multilingual user support.
